# Product

## Register

product

## Users

Operators, admins, and template evaluators who need to manage customers, role access, and dashboard health from a familiar SaaS admin surface.

## Product Purpose

React Admin Dashboard is a production-ready admin template. It demonstrates authenticated routing, customer lifecycle management, dashboard reporting, theme controls, and team role administration with mock APIs that can be replaced by a real backend.

## Brand Personality

Efficient, familiar, and composed. The interface should feel like a dependable work tool, with clear controls and restrained presentation.

## Anti-references

Avoid marketing-page composition, decorative dashboard chrome, invented controls, and visual changes that fight Ant Design's established component vocabulary.

## Design Principles

- Preserve task flow over novelty.
- Make destructive actions explicit and recoverable in intent.
- Show loading, empty, and error states where users would otherwise be guessing.
- Keep density appropriate for repeated admin work.
- Prefer framework-native patterns before adding custom UI.

## Accessibility & Inclusion

Default to accessible Ant Design primitives, keyboard-reachable actions, clear labels, and reduced reliance on color alone for state.
